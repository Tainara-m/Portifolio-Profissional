# ✨ Portfólio Profissional — Tainara Martins

Portfólio criativo moderno com foco em **UI/UX, Front-end e produto digital**.

Desenvolvido com HTML, CSS e JavaScript puro, o projeto aplica conceitos de design system, glassmorphism, navegação responsiva e microinterações para criar uma experiência elegante, moderna e profissional. Houve uso de Inteligência Artificial em algumas etapas do projeto.

---

## 🎯 Objetivo do Projeto

Criar uma interface que represente:

- Clareza visual
- Consistência de design
- Estrutura de produto
- Experiência responsiva
- Identidade visual própria

Este portfólio não é apenas uma vitrine de projetos, mas também uma demonstração prática de:

- Estruturação de layout
- Arquitetura CSS
- Organização de componentes
- UX aplicada em landing pages e produto digital

---

## 🧩 Tecnologias Utilizadas

- HTML5
- CSS3 (Design System + Glassmorphism)
- JavaScript Vanilla
- Font Awesome (ícones)
- Deploy via GitHub Pages

---

## 🖥️ Principais Features

### 🔹 Topbar Centralizada
- Navegação em formato pill
- Itens ativos com destaque visual
- Blur + efeito moderno
- Totalmente responsiva

### 🔹 Design System
- Tokens definidos via `:root`
- Sistema de cores consistente
- Controle de tema claro/escuro
- Componentização visual reutilizável

### 🔹 Responsividade
- Layout adaptado para mobile
- Menu dropdown com overlay
- Grid adaptável

### 🔹 Microinterações
- Hover effects
- Transições suaves
- Scroll reveal com IntersectionObserver

---

## 🎨 Conceito Visual

O projeto utiliza:

- Glassmorphism
- Gradientes sutis
- Efeito de glow
- Bordas suaves
- Ícones padronizados
- Hierarquia tipográfica forte

O objetivo foi criar uma estética moderna alinhada a produtos digitais e interfaces SaaS.
